# 📦 ZFundPilot 项目说明书（个人基金分析系统）

## 一、项目名称

**ZFundPilot（个人基金分析与风险管理系统）**

---

## 二、项目定位（非常重要）

ZFundPilot 是一个**本地运行的个人投资分析工具**，用于：

> 管理基金持仓 + 自动更新净值 + 分析收益与风险 + 提供结构优化建议

⚠️ 不是交易系统，不做自动买卖，不连接券商。

---

## 三、核心目标

系统需要解决四个问题：

### 1）我买了什么？

* 基金持仓记录
* 买入金额 / 时间 / 份额

### 2）现在赚了多少？

* 当前净值自动更新
* 实时/每日收益计算
* 历史收益曲线

### 3）风险有多大？

* 最大回撤
* 波动率
* 仓位集中度
* 资产结构风险

### 4）需不需要调整？

* 结构性风险提示
* 再平衡建议（非交易指令）

---

## 四、核心功能模块

# 1️⃣ 持仓管理模块（Portfolio Module）

### 功能：

* 添加基金持仓
* 修改/删除持仓
* 查看当前组合

### 数据字段：

```text
fund_code      基金代码
fund_name      可选
buy_amount     买入金额
buy_date       买入日期
cost_nav       买入时净值（可选）
```

---

# 2️⃣ 净值数据模块（Data Fetch Module）

### 功能：

* 自动获取基金每日净值
* 更新历史净值曲线

### 数据源：

* AkShare（优先）
* 天天基金数据

### 更新方式：

* 手动触发 + 定时任务（每日）

---

# 3️⃣ 收益计算模块（Return Analysis）

### 功能：

* 单基金收益
* 总组合收益
* 年化收益（IRR）
* 收益曲线

### 核心公式：

```text
收益率 = 当前净值 / 成本净值 - 1
```

---

# 4️⃣ 风险分析模块（Risk Engine）

### 功能：

#### ① 回撤分析

```text
最大回撤（Max Drawdown）
```

#### ② 波动率

```text
收益标准差
```

#### ③ 仓位集中度

```text
最大单基金占比
```

#### ④ 风险提示规则：

```text
如果回撤 < -15% → 高风险提示
如果单基金 > 50% → 集中度过高
```

---

# 5️⃣ 组合结构分析模块（Portfolio Structure）

### 功能：

分析基金组合结构：

* 权益类占比
* 债券类占比
* 行业风格（可估算）
* 地域暴露（A股 / 美股 / QDII）

---

# 6️⃣ 再平衡建议模块（Rebalance Advisor）

⚠️ 注意：不是交易建议，而是结构优化建议

### 输出示例：

```text
⚠ 当前组合风险较高：

- 科技基金占比：72%
- 债券基金占比：5%

建议：
- 降低科技基金暴露
- 增加债券类资产
- 控制单一基金不超过40%
```

---

# 7️⃣ 可视化界面模块（UI Module）

建议使用 Streamlit（最简单）

### 页面功能：

* 组合总览
* 收益曲线图
* 持仓列表
* 风险提示面板

---

## 七、数据库设计（SQLite）

### 表1：持仓表 holdings

```sql
CREATE TABLE holdings (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    fund_code TEXT,
    fund_name TEXT,
    buy_amount REAL,
    buy_date TEXT
);
```

---

### 表2：净值历史 nav_history

```sql
CREATE TABLE nav_history (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    fund_code TEXT,
    date TEXT,
    nav REAL
);
```

---

## 八、数据更新逻辑

### 每日流程：

```text
1. 拉取基金最新净值
2. 写入 nav_history
3. 更新收益计算
4. 更新风险指标
5. 更新UI展示
```

---

* 风险管理
* 组合优化建议
* 长期结构判断

---

## 十一、一句话总结（给AI理解用）

> ZFundPilot 是一个本地运行的个人基金组合分析系统，核心目标是自动更新基金净值数据，计算收益与风险指标，并基于组合结构给出再平衡与风险提示建议，而非交易决策系统。
